# Local Environment
1) Open a terminal.
2) Clone repo: ``` git clone https://github.com/pandaman404/portfolio```
3) Enter to website folder: ``` cd website ```
4) Install dependencies: ``` npm install ```
5) Run local server: ``` npm start ```
6) Go to: ``` localhost:3000 ```
